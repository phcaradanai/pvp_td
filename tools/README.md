# tools/README.md

## Purpose
Contains utilities and scripts that aid development, validation, and build processes.

## What belongs here
- Helper scripts for data validation, CI pipelines, code generation.
- Documentation for tooling usage.

## What must NOT be placed here
- Game source code or gameplay logic.
- Large binary assets.

## Future expected files
- `validation/` sub‑folder with schema validation scripts.
- Build scripts (e.g., `build.ps1`).
