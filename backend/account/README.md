# backend/account/README.md

## Purpose
Manages player authentication, profile data, and account information.

## What belongs here
- User login/registration services.
- Profile retrieval and update logic.
- Secure handling of credentials (hashing, tokens).

## What must NOT be placed here
- Gameplay logic or match simulation.
- UI rendering code.

## Future expected files
- `auth_service.gd`
- `profile_manager.gd`
