# client/creeps/README.md

## Purpose
Contains creep prefab definitions and visual assets.

## What belongs here
- Creep scene files (e.g., `.tscn`).
- Visual resources for creeps (textures, meshes, animations).
- References to data definitions in `data/` for stats.

## What must NOT be placed here
- Gameplay logic for movement, damage, or AI – belongs in backend `match/`.
- Server‑side authority code.

## Future expected files
- `basic_creep.tscn`
- `creep_config.json` (placeholder linking to data definitions).
