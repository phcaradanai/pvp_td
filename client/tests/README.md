# client/tests/README.md

## Purpose
Test suite for client‑side code.

## What belongs here
- Unit tests for pure functions and utilities.
- Integration tests for module interactions (e.g., data loading, UI event flow).
- Mock objects for networking and backend responses.

## What must NOT be placed here
- Server‑side tests or integration with backend services.
- Gameplay logic tests that belong in backend match tests.
